# Security Policy

WikiBrain processes conversations and writes redacted plaintext to a local
vault. Security and privacy bugs in capture scope, redaction, deletion, hook
installation, or recalled-context boundaries are treated as high priority.

## Supported versions

The latest `0.1.x` release receives security fixes while WikiBrain is in alpha.

## Reporting a vulnerability

Do not include a real credential, private conversation, or vault archive in a
public issue. Contact [@hungrytech](https://github.com/hungrytech) privately
before sharing reproduction data. A synthetic fixture that demonstrates the
same behavior is preferred.

For ordinary bugs that contain no private data, use the public
[issue tracker](https://github.com/hungrytech/wikibrain/issues).
